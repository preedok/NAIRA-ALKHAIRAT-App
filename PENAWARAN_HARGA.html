PENAWARAN HARGA

Sistem Terintegrasi (Web Dashboard + Mobile Owner + Landing Page)

Dari: PT INSAN CITA INTEGRASI
Penyedia Jasa Software

Berikut rincian lengkap penawaran harga sewa sistem terintegrasi untuk PT Bintang Global Group (Backend API, Web Dashboard, Mobile App Owner, dan Landing Page terintegrasi seperti halaman aplikasi travel). Harga sewa tersedia untuk periode per bulan, per 3 bulan, per 6 bulan, dan per tahun. Biaya server dicantumkan terpisah: PT Bintang Global Group dapat memilih tanpa server (infrastruktur sendiri) atau dengan server (PT Insan Cita Integrasi yang menyediakan).

1. Rincian Detail per Modul Aplikasi

Setiap modul mencakup fitur-fitur berikut (termasuk dalam semua paket sewa).

No Modul & Rincian Fitur
1 Autentikasi & Manajemen User
  • Login / logout aman; session management; token JWT dengan refresh
  • Role-based access control (RBAC): Super Admin, Admin Pusat, Admin Koordinator, Invoice Koordinator, Visa Koordinator, Tiket Koordinator, Role Hotel, Role Bus, Owner, Role Accounting
  • Manajemen user: tambah, edit, nonaktifkan, aktivasi ulang; assign role, cabang, wilayah; daftar user per cabang/wilayah
  • Reset password (link email), ganti password, kebijakan kata sandi
  • Audit log akses (opsional); pembatasan percobaan login
2 Cabang & Wilayah
  • Master provinsi; master wilayah (group cabang); master cabang dengan alamat lengkap, kontak, timezone
  • CRUD cabang & wilayah; assign user ke cabang/wilayah; hierarki data
  • Filter dan scope data berdasarkan cabang/wilayah per role; restriksi akses per cabang
  • Daftar cabang untuk dropdown/form di seluruh aplikasi
3 Onboarding Owner
  • Registrasi calon owner: data perusahaan, nama penanggung jawab, alamat lengkap, NPWP, email, WhatsApp, wilayah operasional
  • Download template MOU; upload MOU yang sudah ditandatangani; preview dokumen
  • Verifikasi MOU oleh Admin Pusat; approval/reject dengan catatan revisi; notifikasi ke owner
  • Deposit awal: upload bukti transfer, input nominal; verifikasi mutasi bank oleh Admin
  • Assign cabang berdasarkan alamat/wilayah; final approval Admin Cabang
  • Status alur lengkap: Registered → Pending MOU → Pending Deposit → Assigned → Active; tampilan status di dashboard owner
4 Produk & Harga
  • Master produk: Hotel, Visa, Tiket, Bus, Paket (kombinasi layanan); nama, deskripsi, satuan (pax/pack), mata uang
  • Harga general & super promo; aturan DP (min 30% normal, 50% promo); tenggat waktu bayar; aturan penalti
  • Kurs cabang (IDR/SAR); aturan penalti bus; harga hotel musiman (jika diizinkan per cabang)
  • Harga khusus per owner (ajuan Admin Cabang); audit log perubahan harga (siapa, kapan, nilai lama–baru)
  • Product price per branch; filter produk per cabang untuk tampil di order & landing page
5 Order & Invoice
  • Buat / edit / hapus order (Owner, Invoice Koordinator); scope per cabang/wilayah; validasi role
  • Order items: hotel, visa, tiket, bus, handling, paket; kalkulasi total, DP, sisa; multi-currency
  • Validasi bisnis: visa wajib dengan hotel; manifest wajib untuk visa/tiket; bus min 35 pack + penalti otomatis; tenggat DP 3 hari; batal otomatis jika 24 jam belum DP (tentative)
  • Status invoice: Draft, Tentative, Partial Paid, Paid, Processing, Completed, Overdue, Canceled, Refunded; alur status dan notifikasi
  • List & filter order/invoice per role; detail order, item, progres; export list; pencarian
  • Kirim hasil order (send-result) ke owner; lampiran file invoice PDF
6 Pembayaran & Bukti Transfer
  • Upload bukti transfer oleh Owner; input nominal; lampiran file ke invoice; multiple bukti per invoice
  • Verifikasi pembayaran oleh Role Invoice; cross check mutasi, nominal, nama pengirim; approve/reject dengan catatan
  • Reallocasi pembayaran antar invoice (overpayment); riwayat pembayaran per invoice
  • Notifikasi otomatis: DP diterima, lunas, overdue; update status invoice; email & in-app
7 Progres Layanan (Hotel, Visa, Tiket, Bus)
  • Hotel: ketersediaan kamar, booking, status kamar (Available → Booked → Occupied); input nomor kamar jamaah; progres: Waiting Confirmation → Confirmed → Room Assigned → Completed; update harga musiman jika diizinkan
  • Visa: terima manifest; progres dokumen (Received → Submitted → In Process → Approved → Issued); upload hasil (zip/rar); kirim ke Owner otomatis; send-result
  • Tiket: input manifest ke maskapai; cek seat; booking; status Seat Reserved → Ticket Issued; upload tiket; kirim ke Owner
  • Bus: kelola armada & ketersediaan seat; validasi min 35 pack; konfigurasi penalti; update status keberangkatan
8 Refund & Pembatalan
  • Request refund oleh Owner; pengecekan otomatis: hotel/visa/tiket sudah diproses atau belum
  • Approval / reject refund oleh Admin/Invoice; aturan: belum diproses = refund; sudah diproses = deposit non-refundable
  • Status: Requested, Approved, Rejected, Refunded; audit log; notifikasi ke owner
9 Dashboard & Laporan
  • Dashboard per role: rekap order, invoice, owner, hotel, visa, tiket, bus (scope cabang/wilayah); widget & grafik
  • Filter: cabang, wilayah, tanggal (harian/mingguan/bulanan); date range
  • Export Excel / PDF untuk rekap transaksi, laporan cabang, laporan global (Admin Pusat)
  • Monitoring Super Admin: overview sistem, health check
10 Modul Accounting
  • Chart of accounts; laporan keuangan; aging piutang; rekening koran; filter per cabang/wilayah
  • Integrasi Accurate: akuntansi, penjualan, pembelian, persediaan, pajak, kas/bank, aset tetap
  • Export data untuk keperluan pembukuan
11 Notifikasi
  • In-app notification; daftar notifikasi per user; mark as read
  • Email: invoice baru, DP diterima, lunas, overdue, progres hotel/visa/tiket, refund, MOU
  • WhatsApp (jika terintegrasi): notifikasi penting ke Owner & koordinator
12 Mobile App (Owner)
  • React Native (Android): login owner, lihat daftar order & invoice; responsive
  • Detail order, progres hotel/visa/tiket/bus; profil owner; notifikasi push
  • Navigasi: Home, Trip Saya (order/invoice), Paket, Profil; bottom tab
13 Landing Page (Terintegrasi)
  • Konsep: Halaman depan publik (seperti situs travel/umroh) yang terintegrasi dengan data aplikasi; tampil profesional dan siap dipakai untuk branding & lead.
  • Hero section: Headline, subheadline, CTA (Daftar sebagai Mitra / Login); background gambar/video; responsif desktop & mobile.
  • Paket Umroh / Layanan: Tampilan paket atau layanan (Hotel, Visa, Tiket, Paket) yang diambil dari master produk & harga di backend; filter per cabang/wilayah jika perlu; kartu paket dengan nama, harga, deskripsi singkat, tombol “Hubungi” atau “Daftar”.
  • Profil & Keunggulan: Section tentang perusahaan (Bintang Global Group), keunggulan, nilai plus; konten dapat diatur via CMS atau static.
  • Testimonial / Galeri: Optional section testimonial atau galeri perjalanan; data dapat dari backend atau static.
  • Kontak & CTA: Form hubungi kami (nama, email, telepon, pesan); tombol WhatsApp; alamat & kontak; CTA akhir (Daftar Mitra / Login ke Dashboard).
  • Footer: Link (Login, Daftar, Syarat & Ketentuan, Kebijakan Privasi); copyright; sosmed (jika ada).
  • Integrasi teknis: API yang sama dengan dashboard (produk, paket, cabang); link Login mengarah ke aplikasi dashboard; link Daftar ke form registrasi owner; satu domain/subdomain dengan aplikasi atau subdomain terpisah (landing.domain.com).

Rincian per periode

  • Per bulan: Kontrak per bulan; perpanjangan bulanan. Tanpa server: Rp 30.000.000/bulan. Dengan server: Rp 35.000.000/bulan (aplikasi Rp 30 jt + server Rp 5 jt).
  • Per 3 bulan: Bayar sekali untuk 3 bulan. Tanpa server: Rp 87.000.000. Dengan server: Rp 102.000.000 (aplikasi 3 bln + server 3 × Rp 5 jt).
  • Per 6 bulan (Promo): Bayar sekali untuk 6 bulan; harga khusus. Tanpa server: Rp 165.000.000 (≈ Rp 27,5 jt/bulan). Dengan server: Rp 195.000.000 (≈ Rp 32,5 jt/bulan).
  • Per tahun (Promo): Bayar sekali untuk 12 bulan; harga khusus. Tanpa server: Rp 300.000.000 (≈ Rp 25 jt/bulan). Dengan server: Rp 350.000.000 (≈ Rp 29,2 jt/bulan).
